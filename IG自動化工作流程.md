# 品牌知識 IG 自動化發佈流程

## 目標
將每日品牌知識內容自動生成 IG 圖文，發佈到 IG，並在 LINE 官方帳號發送連結，引導追蹤者到 IG 觀看。

---

## 完整工作流程

### 階段一：內容生成（自動）
1. AI 根據「創意世界建構師」品牌知識庫，生成當日主題內容
2. 內容拆分為兩部分：
   - **IG 圖片文字**：精煉的標題 + 3 個重點（適合放在圖片上）
   - **IG 貼文文案**：完整說明 + Hashtag（可直接複製貼上）

### 階段二：圖文生成（需您確認）
3. 根據文字內容，生成 IG 風格圖片（含原創插圖）
   - 尺寸：1080 x 1350（IG 直式貼文最佳比例）
   - 風格：與品牌調性一致（高端品牌顧問感、含原創插圖）
   - 品牌元素顏色與 Logo 標準色一致
4. **圖片生成後，先傳給您確認** ← 您審核這一步
5. 您確認 OK 後，才進入下一步

### 階段三：IG 發佈（自動）
6. 透過 Meta Graph API 將圖片 + 文案發佈到您的 IG
7. 取得發佈後的 IG 貼文連結

### 階段四：LINE 導流（自動）
8. 在 LINE 官方帳號發送訊息：
   - 圖片預覽
   - 簡短引言
   - IG 貼文連結（引導點擊到 IG 觀看完整內容）

## 需要您提供的資訊

| 項目 | 說明 | 狀態 |
|------|------|------|
| IG 帳號名稱 | 您的 IG @帳號 | ⏳ 待提供 |
| IG 帳號類型 | 必須是「商業帳號」或「創作者帳號」 | ⏳ 待確認 |
| Facebook 粉絲專頁 | IG 商業帳號需綁定 FB 粉專 | ⏳ 待確認 |
| Meta Access Token | 自動發佈到 IG 的授權憑證 | ⏳ 待設定 |
| 圖片風格偏好 | 配色、字體、排版風格 | ⏳ 待確認 |
| Logo 標準色 | 品牌主色（用於圖片元素統一） | ⏳ 待提供 |

---

## 技術重點

### IG 自動發佈條件（Meta Graph API）
- IG 必須是 **商業帳號（Business Account）** 或 **創作者帳號（Creator Account）**
- IG 帳號需綁定一個 **Facebook 粉絲專頁**
- 需在 Meta for Developers 建立 App 並取得長期 Access Token
- 圖片需上傳至公開 URL（系統已可用 CDN 上傳）

### 圖片生成規格
- 尺寸：1080 x 1350px（IG 直式）
- 內容：標題 + 3 個重點 + 品牌 Logo/署名
- 含原創插圖，避免使用大蒜相關元素
- 品牌元素顏色與 Logo 標準色一致

### 確認機制
- 圖片生成後必經您確認，確認後才發佈
- 文案與圖片為獨立環節，文案確定後才生成圖片
